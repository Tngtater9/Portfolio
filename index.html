Dev.it

This app was developed to connect businesses with freelancers to help with coding problems. The user can log in and if they represent a business they can search for a freelancer by skill and look at the offers they made. If the user is a freelancer, they can look at all of their offers and accept or deny them. When an offer has been submitted, the business and freelancer can message each other. Dev.it is a full-stack project created over a month with a team following SCRUM methodology.

MAV (Mobile Appointment Viewer)

This app was designed for a mobile auto glass repair service. With, MAV, the user can log in and view appointments for a given day. Numbered markers are used to show in what order the appointments occur. MAV was created as an individual with plenty of constructive feedback from instructors and my mentor. Starting from the main view, the user can click on the "+" to start adding an appointment. The first step lets the user search for the address and the second step lets the user fill in details. If the typical user wants to change the address, they can click on the new appointment marker indicated by the "+" and click, "Change Address." Which takes them to the main screen where they can start over. To view an existing appointment, the user can click on a map marker, and the popup will display minimal details. To view all of the details the user clicks on the link that says, "Details...", in the popup. While viewing the details, the user can delete or modify the appointment.
